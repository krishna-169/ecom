# CampusSwap E-Commerce

CampusSwap is a modern e-commerce platform for students to buy, sell, and swap products easily. Built with HTML, JavaScript, and Supabase for authentication and database management.

## Features
- User authentication (login/signup)
- Product listing and search
- Cart and wishlist management
- Profile editing
- Try-On Studio for virtual product try-on
- Voice search and chatbot

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/krishna-169/ecommerc.git
   ```
2. Open the project in your code editor.
3. Configure Supabase credentials in your JavaScript files.
4. Open `index.html` in your browser to start.

## Files
- `index.html`: Main web application
- `email.js`: Email handling logic
- `supabase-setup.sql`: Supabase database setup
